Sistema para gestion bibliotecaria que permita administrar libros, prestamos y generar reportes estadisticos.

Gestion de libros:
Agregar nuevos libros a un catalogo
Visualiza todos los libros disponibles
Busquedas por titulo del libro o el nombre del autor
Sistema de IDs automaticos y unicos para cada libro

Sistema de prestamos:
Registra prestamos de libros
Devoluciones de libros prestados
Validaciones de disponibilidad de libros
Control de fechas de vencimiento

Reportes y estadisticas:
Conteo total de los libros y su disponibilidad
Distribucion de los libros por genero
Listas de prestamos activos
